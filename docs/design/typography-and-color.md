# Typography and Color

## Typography
- Prefer a more intentional typographic system than a default startup look.
- Maintain strong hierarchy between display, section, card, and UI text.
- Ensure readability for dense marketplace content and dashboards.
- Avoid defaulting to interchangeable sans stacks when the surface would benefit from more authorship.

## Color
- Use semantic roles instead of arbitrary one-off colors.
- Maintain strong contrast and dark-theme-first coherence.
- Distinguish trust, moderation, success, warning, and error states clearly.
- Build atmosphere with layered surfaces, depth, and controlled accent use instead of relying on neon alone.

## Anti-Pattern
- avoid overusing the same neon or blue-accent AI aesthetic without brand logic
- avoid visual noise that weakens comprehension
